# Como executar

- Acesse o simulador [Wokwi ESP32](https://wokwi.com/esp32)

- Copie e cole os códigos fornecidos nessa documentação (arquivo sketch.ino e diagram.json).

- Configure o broker MQTT e tópicos conforme o código.

- Execute e visualize a interação entre sensor, atuador e MQTT.
